Leaflet Location Picker

Simple location picker with Leaflet map

Usage:


<label>Insert a Geo Location 
    <input id="geoloc" type="text" value="" />
</label>

$('#geoloc').locationPicker(function(latlng) {
    alert('position selected: '+latlng)
});
